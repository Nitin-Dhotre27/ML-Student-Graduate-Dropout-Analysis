# Data folder

`DBS.csv` and `DBS_2020.csv` currently contain **synthetic placeholder data**
(generated by `src/generate_synthetic_data.py`) so the whole pipeline is
runnable out of the box.

To train on your real data:
1. Replace `DBS.csv` (training) and `DBS_2020.csv` (testing) with your real,
   semicolon-separated exports, using these exact column names:
   `access, tests, tests_grade, exam, project, project_grade, assignments,
   result_points, result_grade, graduate, year, acad_year`
2. Re-run: `python -m src.train`
3. Restart the backend so it picks up the new `models/dropout_stack_pipeline.joblib`.
